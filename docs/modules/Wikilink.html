    Module Wikilink

    This module serves to parse [[wikilink]] markups into object structures.

    Classes:

    • Wikilink: Encapsulates [[wikilink]] markups with a non-file title as objects. Accessible via Mwbot.Wikilink.
      • ParsedWikilink: A subclass of Wikilink, whose instances are returned by Wikitext.parseWikilinks.
    • FileWikilink: Encapsulates [[wikilink]] markups with a file title as objects. Accessible via Mwbot.FileWikilink.
      • ParsedFileWikilink: A subclass of FileWikilink, whose instances are returned by Wikitext.parseWikilinks.
    • RawWikilink: Encapsulates [[wikilink]] markups with an unparsable title as objects. Accessible via Mwbot.RawWikilink.
      • ParsedRawWikilink: A subclass of RawWikilink, whose instances are returned by Wikitext.parseWikilinks.
